# TechFlow - Digital Solutions

A modern, responsive website for TechFlow, showcasing digital solutions and services. Built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Mobile Menu Toggle**: Hamburger menu for easy navigation on smaller screens
- **Dark Mode Support**: Theme toggle with localStorage persistence
- **Form Validation**: Email and name validation for user signup
- **Smooth Animations**: CSS transitions and effects for enhanced user experience
- **Modern UI**: Clean, professional design with gradient accents

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No additional dependencies or installation required

## Usage

### Navigation
- Click on navigation links to jump to different sections
- Use the hamburger menu on mobile devices for navigation

### Dark Mode
- Toggle dark mode using the theme switch
- Your preference is automatically saved

### Signup
- Fill in the contact form with your name and email
- Form validation provides instant feedback
- Success message displays upon valid submission

## Technologies Used
