# Product Card

A simple product card component displaying a Premium Watch with pricing and purchase options.

## Features

- Product image display
- Product name and description
- Price comparison (current vs. original price)
- Add to Cart button
- Responsive design with CSS styling

## Files

- **index.html** - Main HTML file containing the product card structure
- **styles.css** - CSS styling for the product card layout and appearance
- **watch.jpg** - Product image (referenced in HTML)

## Project Description

This project showcases a clean, modern product card component suitable for e-commerce websites. The card displays:
- Premium Watch product information
- Original price: $199.99
- Current price: $149.99 (discounted)
- Interactive Add to Cart button

## Usage

Open `index.html` in a web browser to view the product card.

## Customization

You can customize:
- Product name and description in the HTML
- Prices and discount amounts
- Styling colors, fonts, and layout in `styles.css`
- Product image by replacing `watch.jpg` with your own image
