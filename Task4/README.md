- **Progress Tracking**: Visual progress bar showing quiz completion status
- **Score Calculation**: Automatic score calculation and percentage display
- **Question Counter**: Shows current question and total number of questions
- **Results Display**: Comprehensive results page with final score and percentage
- **Retake Option**: Ability to restart the quiz and try again
- **Responsive Design**: Clean, modern UI with gradient background

## Getting Started

1. **Clone or Download** the project files to your local machine
2. **Open `index.html`** in your web browser
3. **Start the Quiz** and select your answers
4. **Click "Next Question"** to proceed through the quiz
5. **View Results** after completing all questions
6. **Retake Quiz** to start over

## How to Use

1. Each question displays with four multiple-choice options
2. Click on an option to select your answer
3. The "Next Question" button enables once an answer is selected
4. A progress bar shows your completion status
5. After answering all questions, your score and percentage are displayed
6. Use "Retake Quiz" button to start from the beginning
